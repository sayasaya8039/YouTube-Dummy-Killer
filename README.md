# YouTube Dummy Killer

YouTubeのパクリ動画を検出してオリジナルを表示し、パクリチャンネルをブロックするChrome拡張機能。

## 機能

- **パクリ動画検出**: 動画ページを開くと、タイトルで検索して最も古い投稿を「オリジナル」として特定
- **オリジナル表示**: パクリの可能性がある場合、オリジナル動画へのリンクを表示
- **チャンネルブロック**: パクリチャンネルをブロックして今後非表示に
- **おすすめフィルタリング**: ブロックしたチャンネルの動画をおすすめから非表示

## インストール

### 開発版

1. `bun install` で依存関係をインストール
2. `bun run build` でビルド
3. Chromeで `chrome://extensions` を開く
4. 「デベロッパーモード」を有効化
5. 「パッケージ化されていない拡張機能を読み込む」をクリック
6. `YouTube_Dummy_Killer` フォルダを選択

## 使い方

1. YouTubeで動画を開く
2. パクリの可能性がある場合、右上に通知が表示される
3. 「オリジナルを見る」でオリジナル動画へ移動
4. 「このチャンネルをブロック」でチャンネルをブロック

## 技術スタック

- React 19
- TypeScript
- Vite + CRXJS
- Chrome Extension Manifest V3

## 開発

```bash
# 依存関係インストール
bun install

# 開発サーバー
bun run dev

# ビルド
bun run build

# 型チェック
bun run typecheck
```

## 注意事項

- YouTube内部APIを使用しているため、YouTube側の仕様変更で動作しなくなる可能性があります
- 検索結果の「最も古い動画」をオリジナルと推定しますが、100%正確ではありません
